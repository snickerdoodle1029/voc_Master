# 📊 评论分析智能体 Web版

一个智能的用户评论分析工具，可以自动对评论进行分类、情感打分、紧迫度定级，并生成详细的统计报告。

## ✨ 功能特点

- 🏷️ **自动分类**：将评论自动归类到5大类别（功能稳定性、交互与体验、商业化、内容版权、其他）
- 💯 **情感打分**：基于1-5分标准对评论进行情感分析
- 🚨 **紧迫度定级**：自动判断问题紧迫度（P0高危、P1重要、P2一般）
- 📈 **统计汇总**：生成核心数据汇总表和全量评论明细表
- 📋 **Markdown输出**：支持下载分析报告

## 🚀 快速开始

### 本地运行

1. **安装依赖**
```bash
pip install -r requirements.txt
```

2. **运行应用**
```bash
streamlit run app.py
```

3. **访问应用**
浏览器会自动打开，或者手动访问：`http://localhost:8501`

### 使用步骤

1. 在文本框中输入评论列表（每行一条评论）
2. 点击"开始分析"按钮
3. 查看分析结果（核心数据汇总表和明细表）
4. 可选择下载分析报告（Markdown格式）

## 🌐 部署到云端

### 方法一：Streamlit Cloud（推荐）

1. 将代码推送到 GitHub 仓库
2. 访问 [Streamlit Cloud](https://streamlit.io/cloud)
3. 使用 GitHub 账号登录
4. 点击 "New app"
5. 选择你的仓库和主文件 `app.py`
6. 点击 "Deploy"
7. 获得公开链接，可直接分享给用户

### 方法二：其他平台

也可以部署到其他支持 Python 的平台，如：
- Heroku
- Railway
- Render
- Vercel（需要配置）

## 🎯 分类标准

- **1-功能稳定性**：涉及闪退、卡顿、无法播放、下载失败、网络连接错误
- **2-交互与体验UI/UX**：涉及界面、按钮、操作、字体、夜间模式等
- **3-商业化**：涉及会员价格、广告弹窗、数字专辑购买、自动扣费
- **4-内容版权**：涉及歌曲变灰、版权少、音质差、曲库不全
- **5-其他**：纯情绪宣泄、无意义乱码、好评、非产品相关内容

## 💡 情感打分标准

- **1分（愤怒）**：包含脏话、威胁卸载、极度失望
- **2分（不满）**：提出批评，语气较重，但还在讲道理
- **3分（中立）**：提出建议，或者陈述事实，无明显情绪
- **4分（满意）**：认可产品，但有小建议
- **5分（惊喜）**：纯夸奖，非常喜欢

## 🚨 紧迫度标准

- **P0（高危）**：涉及崩溃、无法使用、资损（钱扣了没到账）
- **P1（重要）**：体验差，但不影响核心功能使用
- **P2（一般）**：视觉建议或新功能请求

## 📝 输出格式

分析结果包含两个Markdown表格：

1. **核心数据汇总表**：包含问题分类、出现次数、情感均分、最高紧迫度、典型槽点
2. **全量评论分析明细表**：包含每条评论的ID、一级分类、情感分数、紧迫度、核心槽点、原文摘要
